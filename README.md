# EosFlutter
Flutter练手项目，仿麦子钱包UI的Eos钱包，

# 项目结构
  * 1.Redux 状态管理
      * EOS
      * 多语言
      * 主题
  * 2.Dio 网络请求
  * 3.shared_preferences，sqflite 数据存储
  * 4.event_bus 事件总线
  * 5.widgets 部件
      * MgxBtnWidget
      * MgxInputWidget
      * MgxItemWidget
      * MgxMaterialWidget
      * MgxTitleBarWidget
